# zenn-dsp-book

Zennで公開しているDSP・音声処理の本とサポートコードのリポジトリです。

## 本のタイトル

**PythonではじめるDSP・音声処理 実践入門**

## 内容

Pythonを使って音声処理を「動かしながら理解する」実践入門書です。

- Ch01: サンプリング定理とエイリアシング
- Ch02: FFTで周波数スペクトルを可視化する
- Ch03: スペクトログラムで音を可視化する
- Ch04: wavファイルを読み込んで解析する
- Ch05: メルスペクトログラム
- Ch06: MFCCで音声を数値化する
- Ch07: 音声分類モデルを作る
- Ch08: スペクトル減算でノイズを除去する
- Ch09: ピッチ（F0）を推定する
- Ch10: ノイズ除去の効果を実験で検証する

## 動作環境

```bash
pip install numpy scipy librosa scikit-learn matplotlib soundfile
```
