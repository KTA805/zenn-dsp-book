## はじめに

ここまでの章では人工的に生成した信号を使ってきました。この章では実際のwavファイルを読み込んで分析します。

「自分が録音した音声に、どんな周波数が含まれているのか」を調べることができます。

---

## 必要なライブラリ

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.io.wavfile import read, write
from scipy.fft import fft, fftfreq
from scipy.signal import stft
```

---

## サンプル音声の準備

手元にwavファイルがない場合は、以下で生成できます。

```python
# 440Hz（ラ）＋880Hz（倍音）のテスト音声を生成
sr = 44100
duration = 2.0
t = np.linspace(0, duration, int(sr * duration), endpoint=False)

signal = (0.6 * np.sin(2 * np.pi * 440 * t) +
          0.3 * np.sin(2 * np.pi * 880 * t) +
          0.05 * np.random.randn(len(t)))  # 少しノイズを混ぜる

signal = (signal / np.max(np.abs(signal)) * 32767).astype(np.int16)
write("sample.wav", sr, signal)
print("sample.wav を生成しました")
```

---

## wavファイルの読み込み

```python
sr, data = read("sample.wav")

print(f"サンプリング周波数: {sr}Hz")
print(f"サンプル数: {len(data)}")
print(f"音声の長さ: {len(data)/sr:.2f}秒")
print(f"データ型: {data.dtype}")
print(f"最大振幅: {data.max()}, 最小振幅: {data.min()}")
```

```
サンプリング周波数: 44100Hz
サンプル数: 88200
音声の長さ: 2.00秒
データ型: int16
最大振幅: 32764, 最小振幅: -32764
```

`int16` は16ビット整数で、値の範囲は-32768〜32767です。計算するときは浮動小数点数に変換しておくと扱いやすいです。

```python
# ステレオの場合は片チャンネルだけ使う
if data.ndim == 2:
    data = data[:, 0]

# 正規化（-1.0〜1.0の範囲に）
data = data.astype(np.float32) / 32767
```

---

## 波形を表示する

```python
t_axis = np.arange(len(data)) / sr

plt.figure(figsize=(12, 3))
plt.plot(t_axis, data, linewidth=0.5)
plt.xlabel("Time [s]")
plt.ylabel("Amplitude")
plt.title("音声波形")
plt.show()
```

波形から読み取れる情報は限られています。振幅の大きさ（音量）と、おおまかな音の長さくらいです。周波数の情報はFFTで初めて見えてきます。

---

## FFTで周波数成分を調べる

```python
N = len(data)
Y = fft(data)
freqs = fftfreq(N, 1/sr)

# 正の周波数のみ、振幅を正規化
mask = freqs >= 0
magnitude = np.abs(Y[mask]) * 2 / N

plt.figure(figsize=(12, 4))
plt.plot(freqs[mask], magnitude, linewidth=0.8)
plt.xlabel("Frequency [Hz]")
plt.ylabel("Magnitude")
plt.title("周波数スペクトル")
plt.xlim(0, 5000)
plt.show()
```

440Hzと880Hzのピークが見えるはずです。生成した音声の構造がそのまま現れています。

---

## 実際の声で確認できること

自分の声を録音して試すと、より面白い発見があります。

```python
# 声のスペクトルを分析する例
sr, voice = read("voice.wav")  # 自分の声を録音したファイル
if voice.ndim == 2:
    voice = voice[:, 0]
voice = voice.astype(np.float32) / 32767

N = len(voice)
Y = fft(voice)
freqs = fftfreq(N, 1/sr)
mask = freqs >= 0
magnitude = np.abs(Y[mask]) * 2 / N

# ピーク周波数を探す
peak_idx = np.argmax(magnitude)
peak_freq = freqs[mask][peak_idx]
print(f"最も強い周波数（基本周波数）: {peak_freq:.0f}Hz")
```

男性の声なら100〜200Hz、女性の声なら200〜400Hz付近に基本周波数のピークが現れます。

---

## 短時間FFTで時間変化を見る

音声は時間とともに変化します。曲全体をFFTするより、短い区間ごとに分析する方が実用的です。

```python
n_fft = 2048
hop = 512

f_stft, t_stft, Zxx = stft(data, fs=sr, nperseg=n_fft, noverlap=n_fft-hop)
mag_db = 20 * np.log10(np.abs(Zxx) + 1e-10)

plt.figure(figsize=(12, 5))
plt.pcolormesh(t_stft, f_stft, mag_db, shading='gouraud',
               vmin=-80, vmax=-20, cmap='viridis')
plt.ylabel("Frequency [Hz]")
plt.xlabel("Time [s]")
plt.title("スペクトログラム（短時間FFT）")
plt.colorbar(label="Magnitude [dB]")
plt.ylim(0, 5000)
plt.show()
```

## 音声の基本情報を一括で表示する

分析をまとめて行う関数を作っておくと便利です。

```python
def analyze_wav(filepath):
    sr, data = read(filepath)
    if data.ndim == 2:
        data = data[:, 0]
    data = data.astype(np.float32) / (2**15 if data.dtype == np.int16 else 1)

    print("=== 音声ファイル情報 ===")
    print(f"ファイル  : {filepath}")
    print(f"サンプリング周波数: {sr}Hz")
    print(f"長さ      : {len(data)/sr:.2f}秒")
    print(f"ナイキスト周波数: {sr//2}Hz")

    N = len(data)
    Y = fft(data)
    freqs = fftfreq(N, 1/sr)
    mask = freqs >= 0
    magnitude = np.abs(Y[mask]) * 2 / N
    peak_freq = freqs[mask][np.argmax(magnitude)]
    print(f"最大ピーク周波数: {peak_freq:.0f}Hz")

    fig, axes = plt.subplots(1, 2, figsize=(14, 4))
    t_axis = np.arange(len(data)) / sr
    axes[0].plot(t_axis, data, linewidth=0.5)
    axes[0].set_title("波形")
    axes[0].set_xlabel("Time [s]")

    axes[1].plot(freqs[mask], magnitude, linewidth=0.8)
    axes[1].set_xlim(0, sr//2)
    axes[1].set_title("周波数スペクトル")
    axes[1].set_xlabel("Frequency [Hz]")

    plt.tight_layout()
    plt.show()

analyze_wav("sample.wav")
```

## まとめ

wavファイルの分析で使う主な処理をまとめます。

| 処理 | 関数 | 得られる情報 |
|------|------|------------|
| 読み込み | `scipy.io.wavfile.read` | 波形データ、サンプリング周波数 |
| 波形表示 | `plt.plot` | 音量・長さの変化 |
| 周波数分析 | `scipy.fft.fft` | 含まれる周波数成分 |
| 時間変化の分析 | `scipy.signal.stft` | 周波数の時間変化 |

次の章では、音声AIでよく使われる特徴量「メルスペクトログラム」を作ります。人間の聴覚特性を取り入れた、より実用的な音声表現です。
